## Specifications

### Progress Indicator

| State | Style |
|-------|-------|
| Completed | Amber-600 filled circle with checkmark |
| Current | Amber-600 border, number inside |
| Upcoming | Slate-300 border, number inside |

### Step Container

| Element | Spec |
|---------|------|
| Max width | 640px |
| Padding | 48px |
| Background | White |
| Border radius | 16px |
| Shadow | lg |

### Navigation

| Button | Position | Style |
|--------|----------|-------|
| Back | Left | Ghost/secondary |
| Continue | Right | Primary (Amber-600) |
| Skip | Left of Continue | Text link |

---

## Interactions

| Action | Result |
|--------|--------|
| Enter valid NPI | Auto-lookup, show practice details |
| Scan QR code | Authenticator adds account |
| Enter valid MFA code | Verify and proceed |
| Click "Skip for Now" | Move to next step |
| Click action card | Navigate to respective feature |

---

## Error Handling

| Error | Display |
|-------|---------|
| Invalid NPI | "This NPI is not valid. Please check and try again." |
| NPI not found | "No practice found for this NPI. You can still continue." |
| Invalid MFA code | "Invalid code. Please check your authenticator and try again." |
| Epic connection failed | "Unable to connect to Epic. You can try again or skip for now." |

---

## Skip Behaviors

| Step | Can Skip | Consequence |
|------|----------|-------------|
| Epic Connection | Yes | Manual claim entry only |
| Practice Profile | No | Required for validation |
| MFA Setup | No | HIPAA requirement |
